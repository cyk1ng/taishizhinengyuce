# MySQL to Oracle SQL 语法转换参考

本文档提供MySQL和Oracle数据库SQL语法的差异对照，帮助将MySQL查询转换为Oracle语法。

---

## 1. 数据类型对照

| MySQL | Oracle | 说明 |
|-------|--------|------|
| `INT` | `NUMBER(10)` | 整数 |
| `BIGINT` | `NUMBER(19)` | 大整数 |
| `FLOAT` | `NUMBER` | 浮点数 |
| `DOUBLE` | `BINARY_DOUBLE` | 双精度浮点 |
| `DECIMAL(10,2)` | `NUMBER(10,2)` | 定点数 |
| `VARCHAR(n)` | `VARCHAR2(n)` | 可变字符串 |
| `CHAR(n)` | `CHAR(n)` | 定长字符串 |
| `TEXT` | `CLOB` | 长文本 |
| `BLOB` | `BLOB` | 二进制数据 |
| `DATETIME` | `DATE` | 日期时间 |
| `TIMESTAMP` | `TIMESTAMP` | 时间戳 |
| `BOOLEAN` | `NUMBER(1)` | 布尔值（0或1） |
| `ENUM('A','B')` | `VARCHAR2(10)` + CHECK约束 | 枚举 |

## 2. 建表语法差异

### 2.1 自增主键

**MySQL:**
```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100)
);
```

**Oracle:**
```sql
-- 方式1：使用序列
CREATE SEQUENCE users_seq START WITH 1 INCREMENT BY 1;

CREATE TABLE users (
    id NUMBER DEFAULT users_seq.NEXTVAL PRIMARY KEY,
    name VARCHAR2(100)
);

-- 方式2：使用IDENTITY（Oracle 12c+）
CREATE TABLE users (
    id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    name VARCHAR2(100)
);
```

### 2.2 索引创建

**MySQL:**
```sql
CREATE INDEX idx_name ON users(name);
```

**Oracle:**
```sql
CREATE INDEX idx_name ON users(name);
```

### 2.3 表注释

**MySQL:**
```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY COMMENT '用户ID',
    name VARCHAR(100) COMMENT '用户名'
) COMMENT='用户表';
```

**Oracle:**
```sql
CREATE TABLE users (
    id NUMBER DEFAULT users_seq.NEXTVAL PRIMARY KEY,
    name VARCHAR2(100)
);

COMMENT ON TABLE users IS '用户表';
COMMENT ON COLUMN users.id IS '用户ID';
COMMENT ON COLUMN users.name IS '用户名';
```

## 3. 常用函数对照

### 3.1 字符串函数

| 功能 | MySQL | Oracle |
|------|-------|--------|
| 字符串连接 | `CONCAT(a, b)` 或 `a || b` | `a || b` 或 `CONCAT(a, b)` |
| 字符串长度 | `LENGTH(str)` | `LENGTH(str)` |
| 子字符串 | `SUBSTRING(str, start, len)` | `SUBSTR(str, start, len)` |
| 大小写转换 | `UPPER()`, `LOWER()` | `UPPER()`, `LOWER()` |
| 去空格 | `TRIM()`, `LTRIM()`, `RTRIM()` | `TRIM()`, `LTRIM()`, `RTRIM()` |
| 字符串替换 | `REPLACE(str, old, new)` | `REPLACE(str, old, new)` |
| 查找位置 | `LOCATE(substr, str)` | `INSTR(str, substr)` |
| 字符串反转 | `REVERSE(str)` | `REVERSE(str)` |
| 左填充 | `LPAD(str, len, pad)` | `LPAD(str, len, pad)` |
| 右填充 | `RPAD(str, len, pad)` | `RPAD(str, len, pad)` |

**示例：**

```sql
-- MySQL
SELECT CONCAT(first_name, ' ', last_name) as full_name;

-- Oracle
SELECT first_name || ' ' || last_name as full_name FROM users;
```

### 3.2 日期函数

| 功能 | MySQL | Oracle |
|------|-------|--------|
| 当前日期时间 | `NOW()`, `CURRENT_TIMESTAMP` | `SYSTIMESTAMP`, `SYSDATE` |
| 当前日期 | `CURDATE()` | `TRUNC(SYSDATE)` |
| 当前时间 | `CURTIME()` | `TO_CHAR(SYSDATE, 'HH24:MI:SS')` |
| 日期加法 | `DATE_ADD(date, INTERVAL 1 DAY)` | `date + 1` 或 `ADD_MONTHS(date, n)` |
| 日期减法 | `DATEDIFF(date1, date2)` | `date1 - date2` |
| 日期格式化 | `DATE_FORMAT(date, '%Y-%m-%d')` | `TO_CHAR(date, 'YYYY-MM-DD')` |
| 字符串转日期 | `STR_TO_DATE(str, format)` | `TO_DATE(str, format)` |
| 提取年份 | `YEAR(date)` | `EXTRACT(YEAR FROM date)` 或 `TO_CHAR(date, 'YYYY')` |
| 提取月份 | `MONTH(date)` | `EXTRACT(MONTH FROM date)` 或 `TO_CHAR(date, 'MM')` |
| 提取日期 | `DAY(date)` | `EXTRACT(DAY FROM date)` 或 `TO_CHAR(date, 'DD')` |
| 星期几 | `DAYOFWEEK(date)` | `TO_CHAR(date, 'D')` |

**日期格式化对照：**

| MySQL格式 | Oracle格式 | 说明 |
|-----------|-----------|------|
| `%Y` | `YYYY` | 4位年份 |
| `%y` | `YY` | 2位年份 |
| `%m` | `MM` | 月份（01-12） |
| `%d` | `DD` | 日期（01-31） |
| `%H` | `HH24` | 小时（00-23） |
| `%i` | `MI` | 分钟（00-59） |
| `%s` | `SS` | 秒（00-59） |
| `%W` | `DAY` | 星期名称 |
| `%a` | `DY` | 星期缩写 |

**示例：**

```sql
-- MySQL
SELECT DATE_FORMAT(created_at, '%Y-%m-%d') as date FROM users;

-- Oracle
SELECT TO_CHAR(created_at, 'YYYY-MM-DD') as date FROM users;

-- MySQL: 获取当前月份
SELECT MONTH(NOW()) as month;

-- Oracle: 获取当前月份
SELECT TO_CHAR(SYSDATE, 'MM') as month FROM dual;
```

### 3.3 数学函数

| 功能 | MySQL | Oracle |
|------|-------|--------|
| 四舍五入 | `ROUND(num, decimals)` | `ROUND(num, decimals)` |
| 向上取整 | `CEIL(num)` | `CEIL(num)` |
| 向下取整 | `FLOOR(num)` | `FLOOR(num)` |
| 绝对值 | `ABS(num)` | `ABS(num)` |
| 取模 | `a % b` | `MOD(a, b)` |
| 幂运算 | `POW(x, y)` | `POWER(x, y)` |
| 随机数 | `RAND()` | `DBMS_RANDOM.VALUE()` |

### 3.4 条件函数

**MySQL:**
```sql
SELECT
    CASE
        WHEN status = 1 THEN '计划'
        WHEN status = 2 THEN '执行中'
        WHEN status = 3 THEN '已完成'
        ELSE '其他'
    END as status_text
FROM dispatch_records;
```

**Oracle:**
```sql
SELECT
    CASE
        WHEN status = 1 THEN '计划'
        WHEN status = 2 THEN '执行中'
        WHEN status = 3 THEN '已完成'
        ELSE '其他'
    END as status_text
FROM dispatch_records;
```

**IFNULL vs NVL:**

```sql
-- MySQL
SELECT IFNULL(name, 'N/A') as name FROM users;

-- Oracle
SELECT NVL(name, 'N/A') as name FROM users;

-- 或使用 COALESCE（两者都支持）
SELECT COALESCE(name, 'N/A') as name FROM users;
```

## 4. 查询语法差异

### 4.1 分页

**MySQL:**
```sql
SELECT * FROM users LIMIT 10 OFFSET 20;
```

**Oracle (12c+):**
```sql
SELECT * FROM users OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;
```

**Oracle (11g及之前):**
```sql
SELECT * FROM (
    SELECT a.*, ROWNUM rn FROM (
        SELECT * FROM users
    ) a WHERE ROWNUM <= 30
) WHERE rn > 20;
```

### 4.2 子查询别名

**MySQL:**
```sql
SELECT * FROM (SELECT * FROM users) as t;
```

**Oracle:**
```sql
SELECT * FROM (SELECT * FROM users) t;
```

注意：Oracle不需要也不支持 `AS` 关键字用于子查询别名。

### 4.3 GROUP BY

**MySQL:**
```sql
-- MySQL允许SELECT非GROUP BY的列
SELECT status, name, COUNT(*) FROM users GROUP BY status;
```

**Oracle:**
```sql
-- Oracle要求SELECT的列必须都在GROUP BY中
SELECT status, MAX(name), COUNT(*) FROM users GROUP BY status;

-- 或使用聚合函数
SELECT status, LISTAGG(name, ', ') WITHIN GROUP (ORDER BY name), COUNT(*)
FROM users GROUP BY status;
```

### 4.4 JOIN语法

**两者基本相同，但需要注意：**

```sql
-- 内连接
SELECT * FROM users u INNER JOIN orders o ON u.id = o.user_id;

-- 左连接
SELECT * FROM users u LEFT JOIN orders o ON u.id = o.user_id;

-- 右连接
SELECT * FROM users u RIGHT JOIN orders o ON u.id = o.user_id;
```

## 5. 约束差异

### 5.1 外键约束

**MySQL:**
```sql
CREATE TABLE orders (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

**Oracle:**
```sql
CREATE TABLE orders (
    id NUMBER DEFAULT orders_seq.NEXTVAL PRIMARY KEY,
    user_id NUMBER,
    CONSTRAINT fk_user FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);
```

### 5.2 CHECK约束

**MySQL:**
```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    age INT CHECK (age >= 0 AND age <= 120)
);
```

**Oracle:**
```sql
CREATE TABLE users (
    id NUMBER DEFAULT users_seq.NEXTVAL PRIMARY KEY,
    age NUMBER,
    CONSTRAINT chk_age CHECK (age >= 0 AND age <= 120)
);
```

## 6. 实际转换示例

### 6.1 简单查询转换

**MySQL:**
```sql
SELECT
    id,
    task_name,
    DATE_FORMAT(created_at, '%Y-%m-%d %H:%i:%s') as formatted_date,
    CASE status
        WHEN 1 THEN '计划'
        WHEN 2 THEN '执行中'
        WHEN 3 THEN '已完成'
        ELSE '其他'
    END as status_text
FROM dispatch_records
WHERE created_at >= '2024-01-01'
ORDER BY created_at DESC
LIMIT 10;
```

**Oracle:**
```sql
SELECT
    id,
    task_name,
    TO_CHAR(created_at, 'YYYY-MM-DD HH24:MI:SS') as formatted_date,
    CASE status
        WHEN 1 THEN '计划'
        WHEN 2 THEN '执行中'
        WHEN 3 THEN '已完成'
        ELSE '其他'
    END as status_text
FROM dispatch_records
WHERE created_at >= TO_DATE('2024-01-01', 'YYYY-MM-DD')
ORDER BY created_at DESC
OFFSET 0 ROWS FETCH NEXT 10 ROWS ONLY;
```

### 6.2 聚合查询转换

**MySQL:**
```sql
SELECT
    status,
    COUNT(*) as count,
    AVG(task_weight) as avg_weight,
    SUM(task_weight) as total_weight
FROM dispatch_records
WHERE DATE_FORMAT(created_at, '%Y-%m') = '2024-01'
GROUP BY status
HAVING COUNT(*) > 0;
```

**Oracle:**
```sql
SELECT
    status,
    COUNT(*) as count,
    AVG(task_weight) as avg_weight,
    SUM(task_weight) as total_weight
FROM dispatch_records
WHERE TO_CHAR(created_at, 'YYYY-MM') = '2024-01'
GROUP BY status
HAVING COUNT(*) > 0;
```

### 6.3 时间范围查询转换

**MySQL:**
```sql
-- 查询今天的数据
SELECT * FROM dispatch_records
WHERE DATE(created_at) = CURDATE();

-- 查询最近7天的数据
SELECT * FROM dispatch_records
WHERE created_at >= DATE_SUB(NOW(), INTERVAL 7 DAY);

-- 查询本月的数据
SELECT * FROM dispatch_records
WHERE DATE_FORMAT(created_at, '%Y-%m') = DATE_FORMAT(NOW(), '%Y-%m');
```

**Oracle:**
```sql
-- 查询今天的数据
SELECT * FROM dispatch_records
WHERE TRUNC(created_at) = TRUNC(SYSDATE);

-- 查询最近7天的数据
SELECT * FROM dispatch_records
WHERE created_at >= SYSDATE - 7;

-- 查询本月的数据
SELECT * FROM dispatch_records
WHERE TO_CHAR(created_at, 'YYYY-MM') = TO_CHAR(SYSDATE, 'YYYY-MM');
```

## 7. 常见错误和修复

### 7.1 字符串连接错误

**错误：**
```sql
-- Oracle中不支持MySQL的CONCAT语法（除非两个参数）
SELECT CONCAT(name, age) FROM users;
```

**修复：**
```sql
-- 使用 || 运算符
SELECT name || age FROM users;

-- 或使用多个参数的CONCAT
SELECT CONCAT(name, CAST(age AS VARCHAR2(10))) FROM users;
```

### 7.2 日期格式错误

**错误：**
```sql
-- Oracle中不支持MySQL的日期格式
SELECT * FROM users WHERE created_at >= '2024-01-01';
```

**修复：**
```sql
-- 使用TO_DATE转换
SELECT * FROM users WHERE created_at >= TO_DATE('2024-01-01', 'YYYY-MM-DD');

-- 或使用日期字面量
SELECT * FROM users WHERE created_at >= DATE '2024-01-01';
```

### 7.3 分页错误

**错误：**
```sql
-- Oracle 11g不支持LIMIT语法
SELECT * FROM users LIMIT 10;
```

**修复：**
```sql
-- Oracle 12c+
SELECT * FROM users FETCH FIRST 10 ROWS ONLY;

-- Oracle 11g
SELECT * FROM (
    SELECT a.*, ROWNUM rn FROM users a WHERE ROWNUM <= 10
) WHERE rn > 0;
```

### 7.4 GROUP BY错误

**错误：**
```sql
-- Oracle要求所有非聚合列都在GROUP BY中
SELECT department, name, COUNT(*) FROM users GROUP BY department;
```

**修复：**
```sql
-- 方法1：添加到GROUP BY
SELECT department, name, COUNT(*) FROM users GROUP BY department, name;

-- 方法2：使用聚合函数
SELECT department, MAX(name), COUNT(*) FROM users GROUP BY department;

-- 方法3：使用LISTAGG（Oracle特有）
SELECT department, LISTAGG(name, ', ') WITHIN GROUP (ORDER BY name), COUNT(*)
FROM users GROUP BY department;
```

## 8. 性能优化建议

### 8.1 索引使用

**两者基本相同：**
```sql
-- 创建索引
CREATE INDEX idx_users_name ON users(name);

-- 创建复合索引
CREATE INDEX idx_users_dept_age ON users(department, age);

-- 删除索引
DROP INDEX idx_users_name;
```

### 8.2 查询提示

**Oracle提供更丰富的提示：**

```sql
-- 使用索引提示
SELECT /*+ INDEX(users idx_users_name) */ * FROM users WHERE name = '张三';

-- 全表扫描提示
SELECT /*+ FULL(users) */ * FROM users;

-- 并行查询提示
SELECT /*+ PARALLEL(users, 4) */ * FROM users;
```

## 9. 快速检查清单

将MySQL SQL转换为Oracle时，检查以下项目：

- [ ] 自增主键改为序列或IDENTITY
- [ ] VARCHAR改为VARCHAR2
- [ ] DATETIME改为DATE或TIMESTAMP
- [ ] ENUM改为VARCHAR2 + CHECK约束
- [ ] NOW()改为SYSDATE或SYSTIMESTAMP
- [ ] DATE_FORMAT改为TO_CHAR
- [ ] STR_TO_DATE改为TO_DATE
- [ ] CONCAT改为 ||
- [ ] IFNULL改为NVL
- [ ] LIMIT改为FETCH FIRST ... ROWS ONLY
- [ ] %Y%m%d格式改为YYYYMMDD
- [ ] 子查询别名去掉AS
- [ ] GROUP BY中的列必须都在SELECT中或使用聚合函数
- [ ] 字符串引号：MySQL支持单引号和双引号，Oracle只支持单引号
- [ ] 注释：MySQL支持#注释，Oracle只支持--和/* */
