# 配网调度业务量智能预测系统 - 前端界面

## 📖 系统简介

这是一个专业的配网调度业务量智能预测系统的前端界面，采用现代化的设计风格，提供流畅的用户交互体验。

## 🚀 快速开始

### 1. 启动后端服务

```bash
# 在项目根目录执行
python src/main.py
```

服务将在 `http://127.0.0.1:5000` 启动。

### 2. 访问前端界面

打开浏览器，访问：

```
http://127.0.0.1:5000
```

或者：

```
http://localhost:5000
```

## 🎯 功能特性

### 💬 智能对话
- 支持自然语言交互
- 实时流式响应
- Markdown 格式渲染
- 代码高亮显示

### 🎯 快速预测
- **今日预测**：预测当天调度业务量
- **7天预测**：预测未来一周业务趋势
- **本月预测**：预测当月业务总量

### 📊 数据可视化
- 业务量趋势图表
- 人员配置建议
- 风险预警展示

### ⚡ 快速操作
- **人员调整建议**：基于预测的人员配置建议
- **风险预警分析**：识别潜在风险点
- **生成决策报告**：生成完整的决策报告

## 🎨 界面说明

### 顶部导航栏
- 显示系统名称和状态
- 提供帮助按钮

### 左侧面板
- **快速预测**：一键预测按钮
- **系统状态**：显示模型和数据源状态

### 中间对话区
- 显示对话历史
- 输入框用于发送消息
- 支持快捷键（Enter 发送）

### 右侧面板
- **预测趋势图**：业务量趋势可视化
- **人员建议**：人员配置建议卡片
- **风险预警**：风险预警信息

## ⌨️ 快捷键

- `Enter` - 发送消息
- `Shift + Enter` - 换行

## 🔧 技术栈

- **前端框架**：原生 JavaScript
- **样式框架**：Tailwind CSS (CDN)
- **图表库**：Chart.js
- **Markdown渲染**：markdown-it
- **代码高亮**：highlight.js
- **API通信**：Server-Sent Events (SSE)

## 🌐 API 接口

### POST /stream_run
流式对话接口

**请求示例：**
```json
{
  "message": "预测未来7天的调度业务量"
}
```

### POST /run
同步对话接口

### GET /health
健康检查接口

### GET /
返回前端首页

## 🔒 环境要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持 ES6+ JavaScript
- 支持服务器推送事件（SSE）

## 📱 响应式设计

界面支持多种屏幕尺寸：
- 桌面端（> 1024px）

## 🐛 问题排查

### 前端无法访问
- 检查后端服务是否启动
- 确认端口 5000 未被占用

### 无法发送消息
- 检查网络连接
- 查看浏览器控制台错误信息
- 确认 API Key 已配置

### 图表不显示
- 检查是否有预测数据
- 刷新页面重试
