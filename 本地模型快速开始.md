# 🚀 本地模型快速开始指南

## 一分钟快速部署

### 1️⃣ 安装 Ollama（首次使用）

**Windows：**
```
1. 访问 https://ollama.com/download
2. 下载并安装
3. 打开命令提示符（CMD）
```

**macOS：**
```bash
brew install ollama
```

**Linux：**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

---

### 2️⃣ 下载模型
```bash
ollama pull qwen2.5:7b
```

**预期输出：**
```
pulling manifest
pulling 8a8a58b38f82... 100% ▕████████████████████▏ 4.5 GB
verifying sha256 digest
writing manifest
success
```

---

### 3️⃣ 启动服务
```bash
ollama serve
```

**预期输出：**
```
couldn't find /root/.ollama/models/manifests
created new database at /root/.ollama/models
Listening on [::]:11434 (version 0.1.35)
```

---

### 4️⃣ 测试模型
```bash
ollama run qwen2.5:7b "你好"
```

**预期输出：**
```
你好！有什么我可以帮助你的吗？
```

---

### 5️⃣ 启动系统
```bash
cd /workspace/projects
python src/main.py
```

**预期输出：**
```
INFO:     Started server process
INFO:     Uvicorn running on http://0.0.0.0:8000
```

---

### 6️⃣ 访问界面
打开浏览器访问：
```
http://localhost:8000
```

---

## 📝 配置文件

### `.env` 文件（关键）
```ini
COZE_WORKLOAD_IDENTITY_API_KEY=ollama
COZE_INTEGRATION_MODEL_BASE_URL=http://localhost:11434/v1
```

### `config/agent_llm_config.json` 文件（关键）
```json
{
  "config": {
    "model": "qwen2.5:7b",
    "temperature": 0.7,
    "top_p": 0.9,
    "max_completion_tokens": 8000,
    "timeout": 600,
    "thinking": "disabled"
  }
}
```

## 🔧 常用命令

### 模型管理
```bash
# 查看已安装的模型
ollama list

# 下载新模型
ollama pull qwen2.5:14b

# 删除模型
ollama rm qwen2.5:7b

# 查看模型信息
ollama show qwen2.5:7b
```

### 服务管理
```bash
# 启动服务
ollama serve

# 查看服务日志
ollama logs

# 停止服务（Ctrl+C）
```

### 测试模型
```bash
# 交互式测试
ollama run qwen2.5:7b
```

## 🐛 故障排除

### 问题 1：无法连接到 Ollama
**症状：** `Connection refused`

**解决方案：**
```bash
# 1. 检查服务是否运行
curl http://localhost:11434/api/tags

# 2. 如果未运行，启动服务
ollama serve

# 3. 检查端口是否被占用
netstat -ano | findstr 11434  # Windows
lsof -i :11434               # Mac/Linux
```

---

### 问题 2：模型未找到
**症状：** `model not found`

**解决方案：**
```bash
# 1. 查看已安装的模型
ollama list

# 2. 下载模型
ollama pull qwen2.5:7b

# 3. 验证模型
ollama show qwen2.5:7b
```

---

### 问题 3：响应速度慢
**症状：** 每秒生成 token 数 < 5

**解决方案：**
```bash
# 1. 检查 GPU 是否被使用（NVIDIA）
nvidia-smi

# 2. 使用更小的模型
ollama pull qwen2.5:3b

# 3. 减少输出长度
# 编辑 config/agent_llm_config.json
# max_completion_tokens: 4000
```

---

### 问题 4：内存不足
**症状：** `CUDA out of memory` 或系统卡顿

**解决方案：**
```bash
# 1. 使用更小的模型
ollama pull qwen2.5:3b

# 2. 使用量化模型
ollama pull qwen2.5:7b-q4_K_M

# 3. 关闭其他占用显存的程序
```
